# Juego Espacio Bebedero - Reto

Este juego fue elegido a partir de varios assets de la biblioteca de Unity. A continuación se listan los recursos utilizados:

### Assets Utilizados:
- **Modelos gratuitos 3D de pokemones**: Se utilizaron modelos en 3D para representar personajes y elementos del juego.
- **Cartoon Cat de Unity**: Asset usado para añadir un gato animado en el juego.
- **CRT - Free**: Efecto aplicado a la cámara para simular una pantalla CRT clásica.
- **StarterAssets**: Configuración de la cámara en primera persona para la experiencia de juego.
- **Tale Monster Cube Env**: Escenario y elementos del nivel utilizando este asset de ambiente.
- **Sonidos**: Se incluyeron efectos como el ladrido de perro y música clásica de *Mario Bros*.
