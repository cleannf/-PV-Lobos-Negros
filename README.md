# Espacio de Bebedero

En este proyecto se llevaron a cabo los retos de programación en Unity 3D. Para su desarrollo, se utilizó un proyecto gratuito de la biblioteca de Unity llamado **Synty Polygon**, el cual sirvió como base para implementar las mecánicas y elementos requeridos.

Además, se creó un video para mostrar el **antes y después** de los cambios realizados en la escena. Puedes verlo haciendo clic en el siguiente enlace:
